O·O·A 첫 글자가 같은 세로선에 오도록 좌측 정렬, 데스크톱 `clamp(5rem, 18vw, 12rem)`, 모바일은 16.7vw로 한 줄 유지. 아웃라인 텍스트("ptimum/pus/rchitecture")는 1.0s 시점에 opacity+translateX로 한 번에 슬라이드 인. 우측 상단 점(intro-dot)은 "건축적 방점" 컨셉으로 텍스트가 다 나온 뒤(2.4s) 등장하고 1초 후(3.4s) 홈으로 전환. 탭 세션당 1회 재생(sessionStorage), 클릭 스킵 가능

## SEO / 공유

- Open Graph / Twitter Card, canonical, JSON-LD(ProfessionalService) 구조화 데이터
- **링크 공유 미리보기(og:image)**: 인트로 화면과 동일한 디자인의 브랜드 이미지 `images/brand/og-cover.jpg`(1200×630)를 전 페이지에 적용. 카카오 공유 디버거에서 캐시 초기화까지 완료
- `robots.txt`, `sitemap.xml` 생성
- favicon 자체 제작(노란 배경 "OOA" + 점) — 실제 로고 있으면 교체 권장. 네이버 검색결과에는 예전 아이콘이 캐시되어 보일 수 있음 → 서치어드바이저에서 재수집 요청 필요
- **구글 서치 콘솔 / 네이버 서치어드바이저 둘 다 등록·소유확인·사이트맵 제출까지 완료**
- GitHub Pages 커스텀 도메인 HTTPS 인증서 이슈 해결(DNS Check 재시도로 해결됨)
- 참고: `ooa-architects.com`(www 없는 루트 도메인)은 DNS 미설정 상태 — 필요시 도메인 등록처에서 A레코드 4개 추가 필요 (185.199.108/109/110/111.153)
