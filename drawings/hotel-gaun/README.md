## 객실 단위평면 디자인 제안 (2026-07-20, 사례 4건 기반)

사용자가 제공한 평면사례 4건에서 추출한 개념을 호텔가은 블록치수에 적용한 6개 대안:

| 안 | 개념 | 참고사례 |
|---|---|---|
| 디럭스 A — 스파인월 | 자유벽 앞 조망향 킹베드, 뒷면 데스크·TV, 좌우 회유동선 | 사례2 (리조트형) |
| 디럭스 B — 분리욕실 | 오픈 더블세면 + WC/샤워 분리, 창측 윈도우시트 | 사례3·4A |
| 디럭스 C — 사선 전실 | 사선 워크인클로젯(쐐기형) + 다이아몬드 라운지 | 사례1 (사선 유닛) |
| 패밀리 A — 알코브 분리 | 슬라이딩 파티션으로 부모/자녀존 분리, 트윈 간격 900 | 사례3·4B |
| 패밀리 B — 스파인월 | 자유벽 양면 더블(창측)+트윈(입구측) 4인 | 사례2·4B |
| 패밀리 C — 좌식 툇마루 | 창측 전폭 평상(+300 좌식존) + 2단침대 | 사례1·2 데크의 실내화 |

전 유닛 통로 치수 기준(사례4: 침대옆 0.8m, 통로 0.9m, 풋존 1.15~1.2m) 검증,
블록치수(디럭스 7.0×5.0m / 패밀리 7.0×6.5m) 유지로 2~4F 층평면에 그대로 교체 적용 가능.

## 4층 층고 상향형 유닛 제안 (2026-07-20)

**전제(가정)** — 계획도면·참고도면 어디에도 단면/층고 정보가 없어 신규 설정한 값입니다.
구조기술사·설비 협의 후 확정이 필요합니다.

- 2~3F 표준 층고 약 3,200mm 대비 **4F만 4,500mm로 상향(+1,300mm)**
- **디럭스(로프트베드)**: 침대만 다락화 — 하부 2,600 + 슬래브 300 + 상부(침대존) 1,600(좌식)
- **패밀리(복층)**: 실 단위 복층화 — 하부 2,200 + 슬래브 300 + 상부(자녀실/침실) 2,000(착석 가능)

각 유닛은 **하부평면 + 상부(로프트/2F)평면 + 개념단면**의 3분할 구성으로 작성했고,
평면 폭(7.0m)은 기존 블록치수와 동일합니다.

| 안 | 개념 |
|---|---|
| 디럭스 로프트 A — 사다리형 | 욕실 위 침대 다락, 하부는 개방형 라운지·워크스루 |
| 디럭스 로프트 B — 계단식수납형 | 침대 진입을 겸하는 계단식 수납 스텝, 하부는 세컨드 라운지 |
| 디럭스 로프트 C — 벙커형(2층침대) | 다락 위에 실제 트윈×2 벙커 배치, 1~4인 유동수요 대응 |
| 패밀리 복층 A — 코너계단형 | 1F 부모존+거실, 2F 자녀 다락실(싱글×2+놀이존), 직선계단 |
| 패밀리 복층 B — 마스터업형 | 1F 리빙+주방, 2F 마스터침실, 대형소파+티테이블 |
| 패밀리 복층 C — 나선계단형 | 1F 거실+주방, 2F 통합베드룸(더블+트윈×2), 나선계단으로 코어 최소화 |

상부평면의 점선 외곽(0LOFT)은 다락/복층 슬래브, 해치존(0OTB)은 오픈보이드(하부 개방)를 의미합니다.

### v1 → v2 변경 (사용자 피드백 반영)

- **패밀리 45.0㎡(저심도형)도 복층으로 전환**: 직선계단 대신 **나선계단(지름 1,200)**을 적용해
  폭 9.0m·깊이 5.0m의 저심도 비례에서도 계단실을 최소 면적으로 확보했습니다. → 패밀리 3실 전체 복층화.
- **통과동선 확보**: 전 유닛에서 진입문→욕실(또는 핵심 동선) 구간은 가구를 배치하지 않고
  비워, 문을 열고 들어와 바로 마주치는 경로에 가구가 걸리지 않도록 했습니다.
- **침대는 정적 공간에 배치**: 진입·주방·계단 동선에서 가장 먼 안쪽 코너(정적 공간)에
  침대를 배치했습니다. 패밀리 복층의 부모 침대도 종전 "소파만 있고 침대가 없던" 누락을
  바로잡아 정식으로 배치했습니다.
- **디럭스 로프트에 간이주방 신설**: 싱크+2구 인덕션 카운터를 통과동선 위쪽 벽면에 밀착
  배치해 있는 그대로 두면서 사다리와 겹치지 않게 재배치했습니다.

### 반영 범위

- **디럭스 로프트 4실(35.0㎡)**: 외측형(①)·내측형(②) — 간이주방·사다리는 벽체 코너 밀착,
  통과동선(y200~1300) 완전 확보, 소파·라운지는 안쪽 정적 공간
- **패밀리 복층 3실**: 코너형(③, 45.5㎡×2) + 저심도형(④, 45.0㎡×1, 나선계단)
  — 부모침실은 안쪽 정적 공간, 계단은 출입문 옆 벽체 밀착
- 욕실·미니주방·출입문 좌표는 기존 2~3F 표준형과 동일하게 유지해 급배수 라이저·복도 연결
  정합성을 확보했습니다.
- 실명·면적은 방 내부에 두면 좁은 공간에서 가구와 겹치므로, 유닛 내부에는 소형 태그(①②③④)만
  표기하고 전체 면적·구성은 도면 하단 **실 스케줄**로 분리했습니다.
- 상부(로프트/2F) 상세 평면과 개념단면은 `hotel-gaun-4f-loft-duplex-units.dxf`를 참고하십시오.
  이 파일은 4층 전체 평면(하부 기준) 반영 결과입니다.

## 층평면 구성 (계획도면 0718 추출 데이터 기준)

건물 외곽은 ㄷ자형: 남측동(x5000~33000 / 깊이 6.5m), 북측동(x0~31500 / 깊이 6.5m),
연결부(x17400~28000)에 홀 + 코어(EV 2대·계단, 6.63×6.3m). 복도 폭 1.5m.

**층별 출입구 위치 (A-Wall Brk 표기 추출):**

- **남측동 (2~4F 동일)**
  - 패밀리룸 45.5 ×2: 복도 단부에서 **측벽(동/서) 코너 출입** (y5400~6200)
  - 패밀리룸 45.0: 복도측 북벽 **중앙 출입** (x17400~18200)
  - 리넨실 25.0: 복도측 북벽 **서단 출입** (x21150~21950)
- **북측동 2F**: 패밀리룸 55.9 → **동측벽 남단 출입**(x8600 벽, y16600~17400) /
  패밀리룸 48.0 ×2 → 남측벽에서 **중앙 휴게공간(자판기) 양측 출입**
- **북측동 3·4F**: 디럭스룸 35.0 ×4 **엇갈림(stagger) 배치** —
  외측 2실은 측벽 출입(복도 단부) + **북측 발코니**, 내측 2실은 휴게공간 양측 출입
  (내측 2실 출입 위치는 도면에 표기가 없어 2F 48룸 패턴 준용)

객실 유닛은 미러 변환(TF)으로 각 실의 출입구·창 방향에 맞게 자동 배치된다.
48.0(9.6×5.0m)·55.9(8.6×6.5m)·45.0(9.0×5.0m) 변형 타입 레이아웃 포함.

## 근거 (입력 도면 분석)

- **호텔가은 계획도면(0718)**: 2F/3F 객실 블록 치수를 그대로 적용
  - 디럭스룸: **7.0m × 5.0m = 35.0m²** (3F 북측 4실)
  - 패밀리룸: **7.0m × 6.5m = 45.5m²** (2F 코너부 주력 타입 / 그 외 45·48·55.9m² 변형)
  - 출입구 폭(A-Wall Brk 800~900), 중복도형 배치, 코어 중앙 집중 반영
- **참고도면 1** (객실유닛 A타입 24.4m² 반복형): 벽체 두께(외벽 200THK / 욕실 칸막이 100THK),
  현관 옆 욕실 배치, 0CONC·0WIN·FUR 레이어 체계, `㎡ /PY` 병기 면적 표기 준용
- **참고도면 2**: 실명 라벨 + 면적 텍스트 표기 방식 준용

## 계획 개요

- **디럭스룸 (기준 2인)**: 현관 — 좌측 욕실(샤워부스·양변기·세면대) — 킹베드(1800×2000)
  + 협탁 2 — TV장 — 창측 라운지체어·티테이블 — 옷장·캐리어받침
- **패밀리룸 (기준 4인)**: 현관 — 좌측 욕실(욕조형) — 더블베드(1800×2000) + 싱글베드(1100×2000) 2
  — 미니주방(싱크·쿡탑·냉장고) — 옷장·TV장 — 창측 좌식테이블
- 창호 W4600(조망 방향), 객실 출입문 W900, 욕실문 W800
- 레이어: `0CONC`(벽체) / `해치벽체` / `0WIN`(창호) / `DOOR`(문) / `FUR`(가구) / `TOIL`(위생기구) / `0TEXT-1`(문자) / `S-DIM`(치수)
