# 호텔 가은 — 객실 단위평면 계획 (디럭스룸 / 패밀리룸)

2026-07-19 세션에서 참고도면 2개 + 호텔가은 계획도면(0718)을 분석해 생성한 객실 단위평면입니다.

## 파일

| 파일 | 내용 |
|---|---|
| `hotel-gaun-guestroom-plans.dxf` | 객실 단위평면 도면 (mm 단위, R2004/AC1018) |
| `hotel-gaun-guestroom-plans.png` | 미리보기 이미지 |
| `gen_rooms.py` | 도면 생성 스크립트 (ezdxf) — 치수·배치 수정 후 재생성 가능 |

## 근거 (입력 도면 분석)

- **호텔가은 계획도면(0718)**: 2F/3F 객실 블록 치수를 그대로 적용
  - 디럭스룸: **7.0m × 5.0m = 35.0m²** (3F 북측 4실)
  - 패밀리룸: **7.0m × 6.5m = 45.5m²** (2F 코너부 주력 타입 / 그 외 45·48·55.9m² 변형)
  - 출입구 폭(A-Wall Brk 800~900), 중복도형 배치, 코어 중앙 집중 반영
- **참고도면 1** (객실유닛 A타입 24.4m² 반복형): 벽체 두께(외벽 200THK / 욕실 칸막이 100THK),
  현관 옆 욕실 배치, 0CONC·0WIN·FUR 레이어 체계, `㎡ /PY` 병기 면적 표기 준용
- **참고도면 2**: 실명 라벨 + 면적 텍스트 표기 방식 준용

## 계획 개요

- **디럭스룸 (기준 2인)**: 현관 — 좌측 욕실(샤워부스·양변기·세면대) — 킹베드(1800×2000)
  + 협탁 2 — TV장 — 창측 라운지체어·티테이블 — 옷장·캐리어받침
- **패밀리룸 (기준 4인)**: 현관 — 좌측 욕실(욕조형) — 더블베드(1800×2000) + 싱글베드(1100×2000) 2
  — 미니주방(싱크·쿡탑·냉장고) — 옷장·TV장 — 창측 좌식테이블
- 창호 W4600(조망 방향), 객실 출입문 W900, 욕실문 W800
- 레이어: `0CONC`(벽체) / `해치벽체` / `0WIN`(창호) / `DOOR`(문) / `FUR`(가구) / `TOIL`(위생기구) / `0TEXT-1`(문자) / `S-DIM`(치수)
