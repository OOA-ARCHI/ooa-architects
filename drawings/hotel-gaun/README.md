## 층평면 구성 (계획도면 0718 추출 데이터 기준)

건물 외곽은 ㄷ자형: 남측동(x5000~33000 / 깊이 6.5m), 북측동(x0~31500 / 깊이 6.5m),
연결부(x17400~28000)에 홀 + 코어(EV 2대·계단, 6.63×6.3m). 복도 폭 1.5m.

**층별 출입구 위치 (A-Wall Brk 표기 추출):**

- **남측동 (2~4F 동일)**
  - 패밀리룸 45.5 ×2: 복도 단부에서 **측벽(동/서) 코너 출입** (y5400~6200)
  - 패밀리룸 45.0: 복도측 북벽 **중앙 출입** (x17400~18200)
  - 리넨실 25.0: 복도측 북벽 **서단 출입** (x21150~21950)
- **북측동 2F**: 패밀리룸 55.9 → **동측벽 남단 출입**(x8600 벽, y16600~17400) /
  패밀리룸 48.0 ×2 → 남측벽에서 **중앙 휴게공간(자판기) 양측 출입**
- **북측동 3·4F**: 디럭스룸 35.0 ×4 **엇갈림(stagger) 배치** —
  외측 2실은 측벽 출입(복도 단부) + **북측 발코니**, 내측 2실은 휴게공간 양측 출입
  (내측 2실 출입 위치는 도면에 표기가 없어 2F 48룸 패턴 준용)

객실 유닛은 미러 변환(TF)으로 각 실의 출입구·창 방향에 맞게 자동 배치된다.
48.0(9.6×5.0m)·55.9(8.6×6.5m)·45.0(9.0×5.0m) 변형 타입 레이아웃 포함.

## 근거 (입력 도면 분석)

- **호텔가은 계획도면(0718)**: 2F/3F 객실 블록 치수를 그대로 적용
  - 디럭스룸: **7.0m × 5.0m = 35.0m²** (3F 북측 4실)
  - 패밀리룸: **7.0m × 6.5m = 45.5m²** (2F 코너부 주력 타입 / 그 외 45·48·55.9m² 변형)
  - 출입구 폭(A-Wall Brk 800~900), 중복도형 배치, 코어 중앙 집중 반영
- **참고도면 1** (객실유닛 A타입 24.4m² 반복형): 벽체 두께(외벽 200THK / 욕실 칸막이 100THK),
  현관 옆 욕실 배치, 0CONC·0WIN·FUR 레이어 체계, `㎡ /PY` 병기 면적 표기 준용
- **참고도면 2**: 실명 라벨 + 면적 텍스트 표기 방식 준용

## 계획 개요

- **디럭스룸 (기준 2인)**: 현관 — 좌측 욕실(샤워부스·양변기·세면대) — 킹베드(1800×2000)
  + 협탁 2 — TV장 — 창측 라운지체어·티테이블 — 옷장·캐리어받침
- **패밀리룸 (기준 4인)**: 현관 — 좌측 욕실(욕조형) — 더블베드(1800×2000) + 싱글베드(1100×2000) 2
  — 미니주방(싱크·쿡탑·냉장고) — 옷장·TV장 — 창측 좌식테이블
- 창호 W4600(조망 방향), 객실 출입문 W900, 욕실문 W800
- 레이어: `0CONC`(벽체) / `해치벽체` / `0WIN`(창호) / `DOOR`(문) / `FUR`(가구) / `TOIL`(위생기구) / `0TEXT-1`(문자) / `S-DIM`(치수)
